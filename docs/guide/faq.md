# 常见问题

## 刷入后抓包仍证书错误？

1. 确认已**重启**
2. WebUI 点「重新注入 APEX」
3. Android 14+ 查看状态页 APEX 是否「已注入」
4. 确认抓包 App 使用的 CA 与模块启用的证书一致（Reqable / ProxyPin / 自定义）

## 设置里看不到系统证书？

部分系统对「可信凭据 → 系统」展示有延迟。可先看 WebUI 状态，或重启后再查。

## 与其它系统 CA 模块一起用可以吗？

不建议。多个模块同时改系统 CA / APEX 挂载容易互相覆盖，请只保留一个。

## WebUI 打不开 / 无法执行命令？

请用 **KernelSU 管理器** 或其它支持模块 WebUI 的客户端打开；普通浏览器没有 `ksu.exec` 桥接。

## 自定义证书文件名报错？

必须是 `^[0-9a-f]{8}\.0$`。PEM/CRT 需先算 hash 再改名。
